# Rite-ly tap

Homebrew tap for [Rite-ly](https://ritely.io).

## Install

```sh
brew install rite-ly/tap/rite
```

Or tap first, then install:

```sh
brew tap rite-ly/tap
brew install rite
```

Or in a `Brewfile`:

```ruby
tap "rite-ly/tap"
brew "rite"
```

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
